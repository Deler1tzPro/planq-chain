# Public Available Endpoints

Below is a list of publicly available endpoints that you can use to connect to the Planq mainnet and public testnets:

You can also use chainlist.org (opens new window) to add the node directly to Metamask.

:::: tabs ::: tab Mainnet

Address Category Maintainer
https://grpc.planq.network Cosmos gRPC Planq Foundation (opens new window)
https://rest.planq.network Cosmos REST Planq Foundation (opens new window)
https://rpc.planq.network Tendermint RPC Planq Foundation (opens new window)
https://evm-rpc.planq.network Ethereum JSON-RPC Planq Foundation (opens new window)
